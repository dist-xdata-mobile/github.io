# 用户手册—数据更新管理系统

作者:Dist 时间： 2020-10-16

# 1. 文档介绍

# 1.1. 文档目的

为了使用户更快的熟悉“数据更新管理系统”的操作方法，规范操作步骤，尽量缩短用户的上手时间，特此编写此用户手册。本手册结合实际操作，配以大量的文字说明和图片展示，确保用户高效快捷的使用本系统。

# 1.2. 读者对象

本文档的预期读者对象包括：

（1）数据更新管理系统相关用户；

（2）项目组成员，包括项目经理、设计人员、开发人员、测试人员等。

# 2. 系统环境

本节描述系统运行所需的软硬件及网络环境。本系统要求环境为工具的最佳适应环境，在要求环境下保障工具的正常运行和服务支持；其他环境下可能出现难以预判的工具问题，只能提供有限的服务支持，无法保障解决所有问题

# 3. 安装指南

# 3.1.1. 安装 ArcGIS ***L****icense* *M****anager***

安装结束后，会自动弹出 License Server Administrator 窗体如下：

License 安装完成后必须先停止服务 ，点击 Start/Stop License Server （若关闭可从开始菜单找到 ArcGIS → License Manager → License Server Administrator）然后如图停止服务。

将破解文件中的 ARCGIS.exe 和 service.txt 拷贝到安装路径 E:\ Program Files (x86)\ArcGIS\License10.5\bin

回到 License Server Administrator 点击启动 Strat，然后再重新读取许可；

2）为“浮动版”产品定义许可管理器，在空格内输入 localhost；

5） 或从系统的开始菜单，找到 ArcGIS、ArcGIS Administrator 进行如上图设置。

6） 点击可用性 Availability→ 刷新 Refesh；

7） 将破解文件中的 AfCore.dll 拷贝到安装路径 C:\ Program Files (x86)\ArcGIS\Desktop10.5\bin

# 5. 系统功能详细说明

# 5.1. 系统管理

# 5.1.1. 连接配置

u 功能说明

配置 SDE 空间库、FTP 文件库的数据库地址，并测试连接通过。

u 操作说明

点击【连接设置】，弹出【连接设置】面板中，包括 SDE 连接、共享目录连接、FTP 连接 3 中连接形式。

SDE 连接中，填写数据库相关内容，并【测试连接】，若测试成功，则点击【新增】按钮，添加成功。同时，可在已建的数据库连接上进行修改，测试通过后点击【修改】按钮，完成修改。

其中：连接名称：自定义名称；服务器名称：空；端口号：sde:oracle11g:IP/orcl；数据库服务名：空；用户名：相应数据库的用户名；

密码：相应数据库的密码。

共享目录连接中，填写共享目录的路径及用户名和密码，并【测试连接】，出现【测试连接成功】即可。同时，可在已建的共享目录连接上进行修改。

# 5.1.2. 历史库设置

u 功能说明

设置现势库和历史库的对应关系。

u 操作说明

点击【历史库设置】，在弹出的对话框中，依次选择现势库及历史库，设置对应关系，确认无误后，点击【确定】，完成设置。右键可添加或删除某一条记录。同时可配置多个现势库及历史库的对应关系。（对应关系一旦设立，不能轻易更改，会影响功能使用）

# 5.2. 资源目录树

u 功能说明

通过资源目录树的形式将数据资源进行整合。以 5 级层次结构进行资源目录树的构建。如图：

u 操作说明

在左侧目录管理框中空白部分右键，在右键菜单中选择【新建目录】，进行【分级】的创建。

在【分级】上右键，选择【新建目录】，可进行目录的创建。

在【目录】上右键，选择【新建目录】，可进行同级目录、子目录、空间库、影像库的创建。

在【空间库】上右键，选择【新建目录】，可进行空间数据子集、空间数据实体的创建。

在【空间数据子集】上右键，选择【添加数据】，可进行空间数据实体的添加。

在【影像库】上右键，选择【添加数据】，可进行镶嵌数据集的创建。

同时，在目录右键菜单中有相应的功能选项，可直接跳转到相应的功能界面中，进行操作。

# 5.3. 更新入库

# 5.3.1. 模板建库

# 5.3.1.1. 模板建库

u 功能说明

根据目标数据库类型，在现势库中创建相应的数据集及数据图层，用于后续数据入库。

u 操作说明

点击【模板建库】，弹出对话框，如下：

其中：

数据模板：本地 Catalog 中创建的 MDB 或 GDB 格式库体结构文件。

目标数据库：下拉选择一个在连接配置中创建好的数据库连接名称。

图层列表：自动加载所选数据模板相对应的数据集及数据图层于文件名列，并匹配出目标数据集和数据图层名。文件名列可进行勾选决定是否创建，目标数据库表名列可进行图层名称修改。

最后，点击【创建图层】按钮，若目标数据库表名列的名称和 SDE 库连接中存在重复，会弹出提示框。如果没有重复，则直接进行模板创建，直至完成。

# 5.3.1.2. ***SDE******修改***

u 功能说明

修改 SDE 现势库，包括图层名称、别名的修改，字段名称、别名的修改，以及字段的新增和删除。同时可以将修改同步到历史库中。

u 操作说明

点击【SDE 修改】，弹出【SDE 修改】弹出框，如下：

目标数据库：下拉选择一个在连接配置中创建好的数据库连接名称。

图层列表中选择需要修改的图层，并点击【下一步】，展示该图层的图层和字段信息。在字段框中空白的地方右键可新增或删除字段。

点击【确定】，当存在删除字段操作时，会弹出提示框，继续执行操作，点击【确定】，完成 SDE 修改。

# 5.3.2. 图层初始化

u 功能说明

对现势库增加系统必须字段及自动创建历史库。

u 操作说明

点击【图层初始化】，弹出界面如下：

在目标数据库中，下拉选择相应的现势库。下方图层列表自动加载所选现势库中的要素集及要素图层。在前面的复选框中勾选需要初始化的图层，点击【确定】，执行初始化。

当所选现势库中已存在初始化过的图层，则图层列表中自动过滤该图层，需勾选下方的【显示全部数据】才能全部在图层列表框中显示。

# 5.3.3. 方案管理

# 5.3.3.1. 创建方案

u 功能说明

根据不同的数据类型或不同的使用习惯创建不同的数据入库方案。

u 操作说明

点击【创建方案】，弹出【新建方案】对话框，如下：

方案名称：自定义方案名称，然后点击【重名检测】，弹出“检测通过！”，继续下一步操作。

入库图层：根据方案内容，勾选相对应的数据集或图层。最后点击【确定】，完成创建。

# 5.3.3.2. 方案管理

u 功能说明

根据创建的方案名称和数据类型，进行方案查询，并可对方案进行查看修改和删除。

u 操作说明

点击【方案管理】，切换到【方案管理】面板，如下：

方案名称：默认为空，可不填或填写内容进行查询。

数据类型：包括矢量数据、栅格数据两类。

点击【查询】按钮，根据筛选条件，加载符合条件的方案列表，展示方案名称、更新图层名称、数据类型、创建人、创建时间等信息。在某一条方案上右键，可对方案进行查看，修改，删除操作。

# 5.3.4. 数据入库

u 功能说明

对于初始入库的数据，可通过数据入库添加到 SDE 现势库图层中。

u 操作说明

点击【数据入库】，弹出【数据入库】弹出框，如下：

任务名称：默认为“数据入库_年/月/日”，点击【重名检测】，弹出“检测通过！”，继续下一步操作。

选择更新包路径：在弹出框中选择本地 MDB 或 GDB 文件。

选择待入库数据：通过展开树形结构，勾选图层，或在选择方案中选择某一方案，下方待入库数据中会自动识别并勾选方案图层。

点击【下一步】，确认源数据与目标数据匹配情况，若出现红色图层名表示该图层没有初始化，需初始化后，才能进行数据入库。

点击【下一步】，点击【入库预览】，在弹出框中，可分屏查看本地 MDB 数据入库前后现势库的数据情况。

确认无误后，点击【开始入库】，进行数据入库，点击【取消】，取消之前操作并关闭弹窗。

# 5.3.5. 按要素更新

# 5.3.5.1. 数据迁出

u 功能说明

将自定义范围中的图层数据迁出为本地 MDB 或 GDB 文件。

u 操作说明

点击【数据迁出】，弹出【按要素更新-数据迁出】弹出框，如下：

任务名称：默认为“数据更新按要素级更新年/月/日”，点击【重名检测】，弹出“检测通过！”，继续下一步操作。

选择迁出范围：有图上绘制、图上选择范围、选择图层三种选择方式。图上绘制为在地图窗口中绘制多边形；图上选择范围为在地图窗口中点选或框选图形；选择图层为加载本地 MDB 图层或 CAD 图层。

选择空间关系：有包含、压盖两种，包含为完全包含在范围中的要素被迁出；压盖为和范围线有压盖的要素均被迁出。

待迁出图层：通过树形结构勾选图层，或在选择方案中选择某一方案，下方待迁出图层中会自动识别并勾选方案图层。

选择迁出格式：支持 MDB 和 GDB 格式。

选择迁出路径：选择本地路径，并命名。

点击【确定】，开始数据迁出，进度条完成后，弹出查看日志提示，可点击【是】查看日志，或点击【否】，直接结束本次数据迁出操作。

若在已有数据迁出任务的范围上继续迁出，会给出提示信息，需先完成迁入任务才能开始新的迁出任务。

# 5.3.5.2. 数据迁入

u 功能说明

将迁出任务中修改好的数据再次迁入，进行要素级数据更新。

u 操作说明

点击【数据迁入】，弹出按要素更新-数据迁入弹出框，如下：

选择迁出路径：选择迁出任务中保存的结果路径。同时，会根据迁入数据信息，自动匹配上任务名称和带迁入数据图层。

点击【下一步】，确认图层匹配信息。

点击【下一步】，点击【入库预览】按钮，在弹出的对话框中可以看到本地数据入库前后的区别。

左侧为入库前现势库中的数据情况，右侧为本地数据入库后现势库中的数据情况。同时，用三种颜色来区分数据的新增、删除、修改状态；其中，黄色表示修改，绿色表示新增，红色网格表示删除。

点击【执行入库】，完成数据迁入。

# 5.3.6. 按范围更新

u 功能说明

根据本地 MDB 的指定图层中的范围线来进行资源目录树中的图层更新。

u 操作说明

点击【按范围更新】，弹出【按范围更新】弹出框，如下：

任务名称：默认为“数据更新按范围更新年/月/日”，点击【重名检测】，弹出“检测通过！”，继续下一步操作。

待入库图层：通过树形结构勾选图层，或在选择方案中选择某一方案，下方待入库图层中会自动识别并勾选方案图层。

更新数据文件：选择本地待更新的 MDB 文件。

更新图层范围：选择本地 MDB 格式的范围图层。

空间关系：有包含、相交两种选择。

点击【下一步】，当更新范围图层中的要素个数超过 1 时，会弹出【提示】框，点击【是】，继续下一步操作。

点击【下一步】，确认源数据与目标数据匹配情况，若出现红色图层名表示该图层没有初始化，需初始化后，才能进行数据入库。

点击【下一步】，点击【入库预览】，在弹出的对话框中可以看到本地数据入库前后的区别。

点击【开始更新】，完成按范围更新入库。

# 5.3.7. 影像入库

u 功能说明

将本地影像文件入库到 SDE 镶嵌数据集和共享文件夹中。

u 操作说明

点击【影像入库】，弹出【影像入库】弹出框，如下：

任务名称：默认为“影像入库_年/月/日”，点击【重名检测】，弹出“检测通过！”，继续下一步操作。

选择目标数据：通过展开树形结构，勾选影像库或其中的某一个镶嵌数据集。

点击【下一步】，点击【+】，选择待导入的栅格数据，支持多数据同时入库。

点击【下一步】，在弹出的界面上，点击【执行入库】，完成影像数据的入库

# 5.3.8. 任务管理

u 功能说明

通过各种查询条件进行任务的查询，并可导出为 EXCEL 格式，进行本地查看。

u 操作说明

点击【任务管理】，弹出【任务管理】面板，如下：

任务类型：包括按范围更新、按要素更新、数据入库、影像入库、数据提取 5 种类型。

# 5.6. 数据库管理

# 5.6.1. 数据提取

u 功能说明

将 SDE 数据库中的数据提取到本地 MDB 或 GDB 中，同时可以对将要提取出来的图层及字段进行筛选。

u 操作说明

点击【数据提取】，弹出【数据提取】弹出框，如下：

任务名称：默认为“数据提取_年/月/日”，点击【重名检测】，弹出“检测通过！”，继续下一步操作。

选择提取范围：有图上绘制、图上选择范围、选择图层三种选择方式。图上绘制为在地图窗口中绘制多边形；图上选择范围为在地图窗口中点选或框选图形；选择图层为加载本地 MDB 图层或 CAD 图层。

选择空间关系：有包含、压盖两种，包含为完全包含在范围中的要素被提取；压盖为和范围线有压盖的要素均被提取。

待提取图层：通过树形结构勾选图层，或在选择方案中选择某一方案，下方待提取图层中会自动识别并勾选方案图层。

字段筛选：鼠标选中某一图层，点击【字段筛选】，在弹出框中勾选待提取的字段，点击【重置】清空勾选内容，点击【确定】完成字段选择，点击【取消】取消操作。

选择保存格式：支持 MDB 和 GDB 格式。

选择迁出路径：选择提取任务中保存的本地结果路径。

点击【确定】，完成数据提取操作。

u 功能说明

# 5.7.1.2. 检测规则

u 功能说明

基于标准规范的控规和专项之间冲突识别规则管理，灵活配置控规和专项之间的冲突识别规则。

u 操作说明

点击【检测规则】，弹出【检测规则管理】弹出框，如下：

在【检查规则列表】处选择新增【+】按钮，在右侧输入【规则名称】、在【规则标准】中选择该规则基于的用地标准，并设定差异比对的级别，点击【自动获取规则项】开始自动生成两个规划之间的比对规则，在差异类型中设定【冲突】或【不冲突】或其他。【颜色设置】是对差异比对出来的不同差异类型进行渲染以及出专题图需要，同一种差异类型颜色需要设置同一种。

# 5.7.1.3. 控专冲突分析

u 功能说明

基于用地标准和检测规则，对控规和专项两个规划之间进行叠加分析，快速识别规划之间的差异分布和差异台账。

u 操作说明

点击【控专冲突分析】，弹出【控专冲突分析】弹出框，如下：

【检测数据 1】和【检测数据 2】选择的为待检测的数据源，【检测字段】为待检测的数据的检测字段，【对应标准】则为该字段对应的数据标准，【结果目录】为差异分析的结果存放的位置，【检测规则】是根据【用地标准】和【检测规则】中的检测规则自动生成的，点击【执行检测】进行控专冲突分析。

执行完毕，会弹出信息提示，点击【是】，加载检测结果到地图窗口中，系统将按照检测规则管理中的规定对数据进行渲染。
