# 1 文档介绍

# 1.1 文档目的

为了使用户更快地熟悉城市设计数字化平台的操作方法，规范操作步骤，尽量缩短用户的上手时间，特此编写操作手册，确保用户高效快捷地使用本系统。

# 1.2 读者对象

本操作手册面向的主要对象是： 使用本系统的用户。

** **

# 2 系统运行环境

# 2.1 硬件环境

为实现流畅的三维效果体验，建议使用如下配置：

名称 要求
CPU 酷睿 i7 或同级别处理器
内存 8 GB 或以上（64 位系统建议 16 GB 以上）
显卡 NVIDIA GTX 580 或以上级别处理芯片，2GB 或以上显存，需要安装显卡驱动
硬盘空间 100GB 或以上 SuperMap 10i（Desktop、iServer）

# 2.2 软件环境

名称 要求
操作系统 Windows 7/8/10
数据库 Oracle 11g R2、Oracle 11g Client、MongoDB、PostgreSQL
其他支撑软件 SuperMap iDesktop 10i、Tomcat 8、JDK 1.8、IIS 7 及以上、.NET framework 4.5.2 及以上

# 2.3 网络类型

100M 网络宽带。

# 4 系统操作说明

# 4.1 系统登录

  • 功能说明

通过系统门户快速访问空间基础沙盘系统、数字化辅助决策系统。因安全管理和权限管理需求，须通过用户名、密码登录系统。

  • 操作说明

双击桌面快捷方式，启动城市设计数字化平台。

输入用户名和密码（使用分配给各处室、各分局的账号），点击【登录】，便可进入系统门户界面。如忘记密码，请联系系统管理员重置。

图 系统登录

进入系统门户界面后，鼠标选择“城市设计空间基础沙盘系统”或“城市设计数字化辅助决策系统”，便可进入相应的系统模块。

图 系统门户

图 城市设计空间基础沙盘系统

图 城市设计数字化辅助决策系统

# 4.2 系统导航

# 4.2.1 修改密码、头像

  • 功能说明

修改当前登录用户的密码、上传或修改用户头像。

  • 操作说明

进入任意系统模块后，鼠标点击系统右上角头像图标，弹出设置面板，然后点击头像旁边的 图标，面板翻转到密码和头像修改界面。

修改密码：根据要求输入旧密码、新密码后，点击【修改密码】按钮即可。

上传/修改头像：点击圆形头像框，选择本地图片文件上传即可。

图 密码修改界面

# 4.2.2 切换系统

  • 功能说明

返回系统门户界面，不注销当前用户，可选择访问其他系统模块。

  • 操作说明

进入任意系统模块后，鼠标点击系统右上角头像图标，弹出设置面板，点击【切换系统】，即可返回系统门户页面，进行系统模块切换。

图 切换系统

# 4.2.3 退出系统

  • 功能说明

注销用户，退出整个系统。

  • 操作说明

进入任意系统模块后，鼠标点击系统右上角头像图标，弹出设置面板，点击【退出】，即可退出系统。

图 退出系统

# 4.3 空间基础沙盘系统

城市设计空间基础沙盘系统汇集基础地理、城市设计、规划编制等丰富多样的数据成果，支持数据查看、数据导出、空间计算、空间测量等功能，实现国土空间一张图从二维平面向立体空间的数字化转型。

# 4.3.1 通用模块

通用模块包括图层管理、空间测量和工具栏三个功能模块。这些功能模块在空间基础沙盘系统和数字化辅助决策系统中都可以使用。

# 4.3.1.1 图层管理

  • 功能说明

图层管理汇集了基础地理库、城市设计二三维库和规划编制空间库这三大类数据资源库，共有超过 70 多个数据图层。通过操作图层管理面板，可以对多源、多尺度的各类数据进行查看、查询。

  • 操作说明

点击系统界面最左侧的展开按钮，展开图层管理面板，点击【数据】或【专题】按钮，可以查看全部数据或城市设计专题数据。

图 图层管理面板

鼠标点击图层名称右侧的，即可在地图区域打开相应图层进行查看。

图 查看图层数据

鼠标点击图层名称右侧的按钮并悬停在该图层名称上方，拖动百分比滚动条可以调整图层的可见度：100%为完全可见。

图 调整图层可见度

查看地图场景的基本操作：

按住鼠标左键，可以平移地图；上下滚动鼠标滚轮，可以放大或缩小地图；按住鼠标滚轮并上下左右移动鼠标，可以旋转地图视角，此时鼠标光标显示为。

# 4.3.1.2 空间测量

空间测量用于在地图场景中以多种方式进行距离和面积的测量。

选择【框选要素】，点击鼠标左键在地图区域绘制多边形，点击鼠标右键完成绘制。面板区域展示所有包含在框选范围内、以及与框选边界相交的要素的属性查询结果。支持多个图层叠加查询。

图 框选要素

# 4.3.1.3.2 全屏截图
  • 功能说明

截取全屏地图（.bmp 格式），保存到本地。只截取地图，不包括功能窗口、信息面板。

  • 操作说明

地图定位到想要保存的视角，鼠标点击工具栏的，选择文件保存路径，点击保存即可。

图 全屏截图样例

# 4.3.1.3.3 全域显示
  • 功能说明

地图定位到威海中心市区全域范围。

  • 操作说明

在任意尺度、视角的地图场景中，鼠标点击工具栏的，地图即会自动定位到威海中心市区全域范围。

图 全域显示范围

# 4.3.1.3.4 图例查看
  • 功能说明

查看当前打开的所有图层的图例。

  • 操作说明

鼠标点击工具栏的，弹出图例面板，可查看图例。再次点击，或点击图例面板右上角的最小化按钮，即可关掉图例面板。

图 图例查看

# 4.3.2 空间计算工具

根据系统内置算法，进行空间计算，包括错落度计算和天际线分析两个分析工具。

# 4.3.2.1 错落度计算

  • 功能说明

错落度计算用于计算一定区域范围内建筑的高低起伏程度，可以辅助用户对城市形态进行考察。

  • 操作说明

点击功能菜单栏按钮，展开菜单，点击【空间计算工具】，再选择【错落度计算】。选择一种查询方式——【点选地块】或【框选范围】。

选择【点选地块】，地图区域默认打开街区范围、现状建筑模型（白模）和规划建筑模型（蓝模）图层。鼠标点选想要查询的地块，再点击【计算】按钮，即可得到该区域内建筑物的错落度值。

图 错落度计算-点选地块

选择【框选范围】，点击鼠标左键绘制多边形，点击右键结束绘制，再点击【计算】按钮，即可得到框选范围内建筑物的错落度值。

图 错落度计算-框选范围

点击【重置】，可取消地块选择、清空计算结果。

# 4.3.2.2 天际线分析

  • 功能说明

在主要街道、观山观海等重要视点位置，系统预设了视廊点。该功能可模拟在视廊点观察到的城市天际线，来辅助城市规划、设计决策。

  • 操作说明

点击功能菜单栏按钮，展开菜单，点击【空间计算工具】，再选择【天际线分析】，打开设置面板。在下拉框中选择【片区】和【视点】，地图自动定位到预设的视点镜头，形成视廊范围和天际线。

图 选择视点、形成视廊范围

可以根据需要修改视点位置坐标（X，Y）、视点高度、视廊的方位角和俯仰角，以及天际线风格，即颜色和显示模式（线或面），系统实时计算渲染当前天际线分析结果。

方位角：观察点相机的方向与正北方向顺时针的夹角，角度范围为：0° 至 360°。

俯仰角：观察点相机方向与地平面的角度，角度范围为：-90° 至 90°。

图 修改视点参数

在地图区域点击鼠标可以从不同视角、不同尺度观察天际线，点击【定位视点】，即可定位到当前设定的视点位置。

点击【清除结果】，即可清空面板上的全部参数设置和天际线分析结果。

点击【导出】，即可导出当前地图屏幕截图，设置文件下载路径，即可保存至本地电脑。

图 天际线截图样例

# 4.3.3 离线数据提取

  • 功能说明

提供各图层数据导出功能，为规划管理者和设计单位提供统一的数据底板。

  • 操作说明

点击功能菜单栏按钮，展开菜单，点击【离线数据提取】，弹出设置面板，可选择查询方式，即【点选范围】、【框选范围】或【上传范围】。

选择【点选范围】，系统默认打开街区范围图层，鼠标点击选择地块范围，并选择需要导出的数据图层。点击【导出】，设置文件路径，即可将数据下载到本地。矢量数据默认导出为 shp 格式，三维模型数据默认导出为 dae 格式。

图 点选范围

图 导出数据样例

选择【框选范围】，点击鼠标左键绘制多边形，点击鼠标右键完成绘制，并选择相应图层，点击【导出】即可导出框选范围内的数据。

图 框选范围

选择【上传范围】，选择本地的范围文件上传（.shp 格式），地图场景加载上传的范围框体，再选择需要下载的图层数据，点击【导出】即可保存到本地。

图 上传范围

点击【重置】即可清除当前全部设置。

# 4.4 数字化辅助决策系统

城市设计辅助决策系统包括设计管控、核提规划要点、建筑方案审查、控规调整论证、方案比选模块以及通用模块，用于解决规划管理实际业务需求。

# 4.4.1 通用模块

参考 4.3.1 节的内容。

# 4.4.2 设计管控

  • 功能说明

通过“总体-单元-街区”三个空间尺度，以及城市“公共空间、形态、风貌、街道、眺望”五大体系，可以结合空间位置直观地查看城市设计数字化成果和管控要求。

  • 操作说明

点击功能菜单栏按钮，再点击【设计管控】，滚动鼠标可切换查看不同空间尺度下的城市设计成果，即总体（市域）、单元和街区。

滚动鼠标缩放至市域尺度，地图区域显示市域尺度对应的广告牌。

图 广告牌

点击广告牌，展开五城体系成果面板，系统默认打开总体城市设计公共空间体系图层，点击右侧【展开】按钮，即可在地图区域打开或关闭相应五城体系成果。

图 总体城市设计范围

**图 总体****城市公共空间体系**

滚动鼠标缩放至单元尺度，地图区域显示单元尺度对应的广告牌。点击广告牌，展开五城体系成果面板，系统默认打开单元城市设计公共空间体系，点击右侧的【展开】按钮，即可在地图区域打开或关闭相应五城体系成果。

图 单元城市设计范围

图 单元城市公共空间体系

图 单元城市眺望体系

滚动鼠标缩放至街区尺度，地图显示街区尺度对应的所有管控规则图标，不同的颜色代表不同的五城体系。

图 规则图标

表 五城体系与规则图标颜色对应关系

五城体系 图标颜色
城市公共空间体系
城市形态体系
城市风貌体系
城市街道体系
城市眺望体系

点选一个街区，展开五城体系规则面板，显示当前街区对应的所有五城体系管控规则。鼠标点选规则面板上的其中一个体系，信息面板自动翻转，显示被选中街区在该城市设计体系下的管控规则以及对应的规则图示，同时地图上对应的规则图标呈被选中状态。

图 街区管控规则

图 管控规则和规则图示

点击面板右上角的按钮，可返回至面板正面。也可以通过点击街区范围内的规则图标，面板会联动显示该图标对应的规则和图示。

# 4.4.3 核提规划要点

  • 功能说明

可以在进行土地出让时，通过平台自动生成某地块的规划设计条件通知书（控规+城市设计要求），给到相关设计单位。

  • 操作说明

点击功能菜单栏按钮，再点击【核提规划要点】，展开设置面板，选择一种范围查询方式——【点选范围】、【框选范围】或【上传范围】。

选择【点选范围】，系统默认打开街区范围图层，鼠标选中需要核提要点的地块，系统自动查询该地块对应的控规要求和城市设计要求。

图 点选范围

点击【框选范围】，街区范围图层关闭，点击鼠标左键在地图区域绘制多边形，点击鼠标右键完成绘制。系统自动查询该框选范围内、以及与该框选范围边界相交的所有街区，并调出这些街区的控规要求、城市设计管控要求。

图 框选范围

点击【上传范围】，选择并上传本地范围文件（.shp 格式），系统自动查询该范围包含和相交的街区所对应的控规要求、城市设计要求。

图 选择范围上传

图 核提范围对应要求

点击【导出】，即可将查询结果生成为报告，以 word 格式下载到本地，可进行编辑。点击【重置】，即可清除当前选择的地块和查询结果。

图 导出报告

图 报告样例

# 4.4.4 建筑方案审查

  • 功能说明

面向分局，根据城市设计管控要求对设计单位提供的建筑设计方案进行技术审查，包括计算机审查和人工审查，并可自动生成审查报告。

  • 操作说明

点击功能菜单栏按钮，再点击【建筑方案审查】，场景中自动打开规划建筑模型（蓝模）。点击【新建项目】，输入项目名称，并点击【确定】。

图 新建项目

点击【导入方案】按钮，打开本地文件路径，将建筑设计方案（.udbx 格式）上传服务器。点击按钮，将建筑方案模型加载至当前场景。

图 方案加载

点击按钮，即可从服务器上删除该项目或方案。

在搜索框中输入项目或方案名称关键词，点击【搜索】按钮，下方列表中即会显示名称中包含该关键词的所有项目以及方案。点击【重置】，即可清空搜索条件，显示全部项目和方案。

点击【开始审查】，进入建筑方案审查模块，包括【方案文本审查】、【城市设计审查】和【规划指标审查】。

# 4.4.4.1 方案文本审查

  • 功能说明

用于查看、审核按照上报规范提交的各类文件，包括方案说明、经济技术指标、采光分析报告等文件。

  • 操作说明

点击【方案文本审查】，显示待审查的文件目录。鼠标单击文件名，即可对文档、图片、CAD 文件等进行查看。点击，可在多文件之间进行切换浏览。

图 方案文本审查

其中 CAD 文件可以在线预览，也可以通过点击按钮，跳转到 CAD 软件，直接在软件中打开文件。

图 CAD 图件查看

在查看文件的过程中，用户可以在下方的意见栏里填写审查意见，点击【保存】即可。

图 文件查看、意见填写

# 4.4.4.2 城市设计审查

基于城市设计管控规则，对建筑设计方案进行精细化审查，主要分为自动审查、人工审查和预设审查三个部分。

# 4.4.4.2.1 自动审查
  • 功能说明

系统自动审查城市设计管控要求中由计算机审查的部分。

  • 操作说明

点击【自动审查】，系统根据该地区配置的城市设计管控规则进行审查计算，审查结果按逐条规则进行统计汇总：红色表示不符合刚性管控要求，橙色表示不符合弹性管控要求，绿色表示符合管控要求。

点击按钮，打开意见填写窗口，鼠标左右拖动管控规则下方蓝色的滚动条，可以查看规则详情。可以对自动审查结果添加具体审核意见，点击【保存】后，关掉窗口即可。再次点击按钮，可对审查意见进行编辑修改。点击【意见查看】，可以在弹出的面板中查看各用户针对自动审查结果填写的意见。

图 自动审查-意见填写

点击管控要求右下方的【xx 条详情】可查看每条规则的详细审查信息，每条详情显示了符合或不符合管控要求的原因以及对应要素的 ID。点击每条详情，即可在地图区域定位到对应的要素对象。

图 自动审查-定位到要素对象

点击建筑顶端的【建筑气泡】按钮，弹出信息框，可查看该对象的所有审查结果。点击按钮，该按钮变为，鼠标可自由拖动信息框。

图 自动审查-查看单栋建筑审查结果

# 4.4.4.2.2 人工审查
  • 功能说明

部分城市设计管控要求是需要人工进行判断的。此模块下，用户可以对建筑设计方案是否满足这些管控要求进行审查。

  • 操作说明

点击【人工审查】，按规则逐条进行审查并填写审查意见。点击按钮，弹出【意见填写】面板，选择是否符合管控要求，并填写审查意见，点击【保存】后关闭窗口即可。

系统根据审查结果自动标记规则状态：红色表示方案不符合刚性管控要求，橙色表示不符合弹性管控要求，绿色表示符合管控要求。再次点击按钮，可对审查意见进行编辑修改。

图 人工审查-意见填写

# 4.4.4.2.3 预设审查
  • 功能说明

预设审查用于人工判断在系统预先设定的视点上往特定方向、角度看，是否满足城市设计眺望体系的管控要求。

  • 操作说明

点击【预设审查】，按规则逐条进行人工审查并填写审查意见。点击信息面板上的某条管控要求，系统自动定位到该管控要求相对应的视点和观察角度。点击按钮，弹出【意见填写】面板，选择是否符合管控要求，并填写审查意见，点击【保存】后关闭窗口即可。

系统根据审查结果自动标记规则状态：红色表示方案不符合刚性管控要求，橙色表示不符合弹性管控要求，绿色表示符合管控要求。再次点击按钮，可对审查意见进行编辑修改。

图 预设审查

# 4.4.4.3 规划指标审查

用于审查建筑设计方案的指标是否符合控规、城市设计以及出让技术指标的相关要求。

# 4.4.4.3.1 规划指标
  • 功能说明

用于查看方案所在街区的控规要求指标和城市设计建议指标。

  • 操作说明

选择【规划指标审查】下面的【规划指标】，即可查看控规方案所覆盖的各街区的控规要求指标和城市设计建议指标。点击【控规要求指标】或【城市设计建议指标】的小标题，可以收起或展开标题下方的指标内容。

图 规划指标查看

# 4.4.4.3.2 指标对比
  • 功能说明

用于人工审查建筑设计方案的指标是否符合出让技术指标的要求。

  • 操作说明

点击【规划指标审查】下面的【指标对比】，可以查看出让技术指标和建筑设计方案指标。点击每个指标后面的 按钮，弹出【意见填写】窗口，可输入审查意见并选择“符合”或“不符合”管控要求。点击【保存】后关闭窗口，如果审查结果是“符合管控要求”，那么在表格内【审查结果】一栏，图标会变成绿色；如果是“不符合管控要求”，则图标会变成红色。

图 指标对比-意见填写

图 指标对比结果

# 4.4.4.4 意见查看

  • 功能说明

将每个审查功能模块下的所有用户意见分别整合在一起，供用户查看审查意见记录。

  • 操作说明

在任意审查功能模块下，点击信息面板下方【意见查看】按钮，弹出意见查看面板，可以查看所有用户针对该项审查方法的审查记录。比如当前位于【规划指标审查】功能模块，此时打开的意见查看面板显示的是针对【规划指标审查】的全部审查意见。

图 意见查看

# 4.4.4.5 报告生成

  • 功能说明

系统自动将【方案文本审查】、【城市设计审查】和【规划指标审查】的全部审查结果和审查意见生成报告，并支持下载到本地。

  • 操作说明

点击【报告生成】，系统根据审查结果自动生成建筑方案审查报告，可进行在线预览。点击下载 pdf 格式报告，也可以点击最右上角下载 word 格式到本地进行修改。

图 报告预览

# 4.4.5 控规调整论证

  • 功能说明

面向规划管理科室，当局部区域的控规用地需要调整时，通过平台，可以对调整后的控规方案进行基础信息查看和技术论证，包括基本信息分析、规划影响分析和设施影响分析。

  • 操作说明

点击功能菜单栏按钮，再点击【控规调整论证】，展开功能面板。点击【新建项目】，输入项目名称，并点击【确定】。

图 新建项目

点击【导入方案】按钮，打开本地文件路径，将控规调整方案（.udbx 格式）上传服务器。点击方案名称前面的按钮，下拉展示该调整方案的概要。点击按钮，将方案模型加载至当前场景。

图 加载方案

点击按钮，即可从服务器上删除该项目或方案。

在搜索框中输入项目或方案名称关键词，点击【搜索】按钮，下方列表中即会显示名称中包含该关键词的所有项目以及方案。点击【重置】，即可清空搜索条件，显示全部项目和方案。

点击【开始审查】按钮，进入业务分析窗口，即可对该控规调整方案进行相应的分析。辅助分析具体分为三个模块，即【基本信息分析】、【规划影响分析】和【设施影响分析】。点击按钮，窗口缩小为，可避免窗口对地图场景产生遮挡；再次点击可以还原窗口。

# 4.4.5.1 基本信息分析

  • 功能说明

主要用于查看控规调整方案的指标详情、方案区位及周边条件、以及多规合一与城市设计要求。

  • 操作说明

点击【基本信息分析】下面的【方案概况】，查看调整后控规方案各地块的指标。

图 方案概况

点击【基本条件】，查看该控规方案周边的区位条件、现状建设等情况。点击按钮，可选择打开或关闭相应图层。支持同时打开多个图层。通过对地图进行平移、缩放、旋转等操作，选择最佳观看视角。

图 控规用地基本条件

选择【规划要求】，查看该控规方案对应的“多规合一”控制线和城市设计等相关规划要求。点击按钮，即可在地图区域打开相应图层，再次点击该按钮，即可关闭当前图层。

图 规划要求

点击按钮，窗口翻转，即可查看每个专题对应的具体规划要求说明；点击窗口左上角的，可退回到【规划要求】界面：

图 规划要求说明

# 4.4.5.2 规划影响分析

  • 功能说明

主要包括多规分析、城市设计分析和对比分析。

多规分析：计算机自动判断方案是否与三条控制线相冲突。

城市设计分析：对比方案指标与城市设计建议指标。

对比分析：控规调整前后方案指标对比查看。

  • 操作说明

选择【多规分析】，系统自动分析该方案与“多规合一”控制线的冲突情况，即是否压盖生态保护红线、是否压盖永久基本农田、是否超出城镇开发边界。点击控制线标题栏，展开各地块具体的分析结果。点击结果记录，地图区域定位并高亮显示相应压盖地块。

图 多规分析

选择【城市设计】，系统自动分析该控规方案是否满足其所覆盖的街区的城市设计建议指标，并显示对比表格。如果该街区没有城市设计建议指标，则表格中此栏为空：

图 城市设计指标对比表

选择【对比分析】，系统自动分析控规调整前后各项指标对比分析情况，同时地图区域左右分屏显示调整前后的控规方案。调整后的控规方案中，其指标与调整前不同的，以黄色高亮显示。点击结果面板右上角的按钮，可查看展开后的表格面板。左右拖动下方滚动条，可查看完整的表格数据。

图 对比分析

图 表格展开
